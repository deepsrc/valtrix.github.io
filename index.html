Product Overview

Sting, the flagship product of Valtrix Systems, is a bare-metal system validation tool for complex client/server SoCs. Implemented in an architecture agnostic manner, it currently supports AArch64/ARMv8 CPU and PCI express system architectures. Test content reuse is enabled by shift-left compliance and target configuration independent test generation. The fresh and innovative approach to system validation makes it a highly potential tool-of-choice for any SoC implementation.
